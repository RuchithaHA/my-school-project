# USER MANUAL — Greenwood International School

## For parents and visitors

### Home page

Open the website root. You will see a short introduction to Greenwood International School, key features, and a **weather** panel for Bengaluru. Use the navigation bar to go to other sections.

### Apply for admission

1. Open **Admission** (or the apply call-to-action on the home page).
2. Complete the form:
   - **Required:** Student name, date of birth, gender, class applying, father name, mother name, parent email, parent phone, address, city, pincode.
   - **Optional:** Previous school, alternate phone, parent occupation, medical conditions, how you heard about us.
3. Press **Submit**.
4. If anything is missing or invalid, red messages appear next to the fields. Fix them and submit again.
5. On success, a **modal** shows your **application number** and a welcome message. Save the application number for your records. The form clears for a new entry.

### Check application status

1. Open **Application status**.
2. Enter the **application number** exactly as shown on your confirmation (for example `GIS-2026-0001`).
3. Submit to see the current **status** and basic details.

### Contact the school

1. Open **Contact**.
2. Enter your name, email, and message.
3. Submit. Staff can read your message in the admin area.

## For administrators

### Sign in

1. Open **Admin**.
2. Enter password: `admin123` (change process: redeploy with different gating if you need stronger security).
3. After login you will see the dashboard.

### Dashboard overview

- **Summary cards** show totals for all applications and for each status (pending, approved, rejected, waitlisted).
- **Seats** table lists each class with total, booked, and available seats. You can adjust **total seats** for a class when policy changes.
- **Admissions** table lists every application with search and status filter.
- **Actions:** change **status** from the row dropdown; **delete** removes an application (you will be asked to confirm).

### Contacts tab

Switch to the **Contacts** tab to read messages sent from the public contact form.

### Tips

- Use **search** to find a student by name or by application number.
- Use the **status** filter to focus on pending items.
- After changing total seats, confirm that **available** counts look reasonable relative to booked seats.
